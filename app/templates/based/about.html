About GoGiAlign

GoGiAlign is a web-based tool designed for creating, visualizing, and analyzing file alignments.

Features

  • Pairwise file alignment for quick comparisons.
  • Multiple file alignment with phylogenetic tree generation.
  • Detailed history tracking for previous alignments.
